Solutions for [Advent of Code](https://adventofcode.com/) programming puzzles, done each December as a mix of fun and learning.

The code is not meant to be pretty or production-grade — it's about enjoying puzzles and picking up new tools along the way.

## Years

- **2021** — First AoC, combined with learning Haskell. First few days also done in JS on Observable notebooks.
- **2022** — Switched to TypeScript (ts-node). Stopped halfway through — had to pack for a move.
- **2023** — Back to plain JS (node), taking it easy. Still carried over some FP ideas from the Haskell year.
- **2024** — TypeScript again, this time with Deno 2 and Zed REPL (Jupyter-style notebooks). Built a small helper library for faster solving. Also did some of AoC 2016 as prep.
- **2025** — Same Deno + Zed workflow, also tried a shortened format.
