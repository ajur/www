A clone of the old DOS game *Vivi Vicious Viper*, recreated from memory on the PICO-8 fantasy console. This was the first PICO-8 project — after a few days of tinkering, it became something playable and satisfying.

As for the original Vivi - it's the game i was playing as a kid on my first pc. Funny thing is, after trying to implement everything based only on those memories, I've made some research and somehow found it [somewhere in the Internet Archive](https://archive.org/details/ViviViciousViperV2.011990SpareTimeSoftwareAction) :)

The final version (1.0) added persistent high scores and some code cleanup.
