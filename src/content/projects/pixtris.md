Very simple Tetris clone built with Pixi.js. Originally created as a development task for a job interview. One of the first (if not *the* first) games I made with [Pixi.js](https://pixijs.com/).

The blocks art style was inspired by the fantastic pixel bead works of [VGPerlers](https://www.etsy.com/shop/VGPerlers). Some basic implementation ideas came from [Javascript Tetris](http://codeincomplete.com/posts/javascript-tetris/) on Code inComplete.

## Changelog

- **v1.0** — Small improvements and pause menu added.
- **v0.3** — Changed visual style, added some JSDocs.
- **v0.2** — Reorganized and partially rewritten. Ugly, but working.
- **v0.1** — First iteration, single file, not functional.

## Tech Stack

- **Language:** JavaScript
- **Engine:** Pixi.js
- **Bundler:** Webpack
